What's new in GGiS

Posted by: Caleb Rowan, Posted on: 1 August 2023

We've made some changes to the system in the following areas:

  • Home
  • Reports (Now called Downloads)
  • View Schemes and View Grants in Aid
  • View Awards and Associated Awards
  • Previous Uploads and Associated Documents
  • What’s new page

Read about all the changes below, and read on to find out about upcoming changes.

Home

We have changed the homepage, introducing text explaining each section to make it easier for new users. The coloured tiles have been removed.

You can now navigate directly to the different sections under Grant Schemes and Grant Awards using quick links.

We introduced a notification banner to update you on any changes to the system.

Reports (Now called Downloads)

We have retired the old Reports feature because it was prone to errors and did not meet our users’ needs.

We have replaced it with a feature called Downloads which includes options for you to download data that was previously accessible through Reports.

You can then do further filtering, analysis and visualisation using Excel or another tool on your computer.

We have removed the How to video for using Reports and Dashboards because it is no longer accurate.

View Schemes and View Grants in Aid

You can now use multiple enhanced filters to further refine your results.

We have added a Download results button that allows you to download a CSV of the results of your search and filters.

We have introduced the option to Change View to see more details about your search results.

We have rewritten the content on this page to make it more helpful.

View Awards and Associated Awards

Improved content for View Awards with clearer options for searching your awards. As with View Schemes and View Grants in Aid, new view, filter and download options are available.

Associated Awards has been changed to match the style and functionality of View Awards.

View Recipients and viewing a single recipient

View Recipients has changed to match the style of the other View pages. Summary information on number and value of grants recipients have been awarded is now displayed.

The page layout for viewing a single recipient has changed so you no longer need to navigate between tabs.

You can now download a list of all awards to that recipient.

Previous Uploads and Associated Documents

We have changed the design of these pages to match the styles elsewhere, the functionality remains the same.

What’s new page

We have added a What’s new page (that’s here!) to inform users of changes to GGiS. Users will be able to see the latest release notes on the front page and access archived notes below.
