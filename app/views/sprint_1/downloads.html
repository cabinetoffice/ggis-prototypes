Downloads

Here are some of the data files you are able to download. You can also download filtered down files on the “View Schemes”, “View Awards” and “View Recipients” pages, as well as within each Recipient record.

Report name

  • User Licences

  • Download a file listing all active users in your department and what type of GGiS account they have, and all departments’ GGIS Grants Champions. These reports are only visible to GGIS Grants Champions.

  • My Organisation’s Schemes

  • Download a file listing your organisation’s grant schemes.

  • All Schemes

  • Download a file listing all grant schemes on GGiS. You cannot view any data from other departments that has been marked as not for publishing for sensitivity/national security reasons.

  • My Organisation’s Awards

  • Download a file listing your organisation’s grant awards.

  • All Awards

  • Download a file listing all grant awards on GGiS. You cannot view any data from other departments that has been marked as not for publishing for sensitivity/national security reasons.
